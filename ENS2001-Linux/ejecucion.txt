Con el objetivo de que el alumno pueda crear un ejecutable del programa ensamblador para  la arquitectura de su maquina, se pone a su disposicion la carpeta fuentes, que contiene los ficheros fuentes del programa ensamblador.

El alumno debera hacer uso del fichero makefile contenido en la carpeta fuentes para crear su propio ejecutable. Para ello:

- Abrira una terminal
- Se ubicara en la carpeta fuentes
- Ejecutara el comando make

Una vez ejecutado este comando, se creara el archivo ejecutable del programa ensamblador con el nombre ens2001.

Cabe destacar que la version para Linux no dispone de interfaz grafica.
